# 桌面日历浮窗

启动：

- 双击 `Start-DesktopCalendar.cmd`
- 开机自启动：双击 `Enable-Startup.cmd`，只需要运行一次
- 安装版用户也可以通过桌面或开始菜单快捷方式启动

显示内容：

- 左侧显示月份、日期、星期；鼠标停在左侧日期上会显示年份，移开后隐藏
- 毕业倒计时
- 法定节假日提醒：没有节假日、且三天内没有节假日时自动隐藏

操作：

- 拖动小卡片空白处可以移动主界面
- 点击右上角 `-` 收起到屏幕右边框
- 右键主界面打开功能菜单，菜单默认贴在浮窗右侧，不遮挡浮窗
- 菜单顶部是背景：单色、混色、天气背景
- 天气背景带轻量动画：晴天阳光、雨天雨滴、雪天雪点，其他天气也有细节动效
- 菜单下方可以缩小到右侧小浮窗、关闭浮窗显示
- 关闭浮窗显示只退出当前界面，不删除程序文件；之后仍可双击启动脚本重新打开
